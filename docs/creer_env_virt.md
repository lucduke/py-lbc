# Créer un environnement virtuel

## Utilisation de venv

Pour créer un environnement virtuel avec le module `venv`, utilisez la commande suivante :
```bash
python -m venv .venv
```

Puis, activez l'environnement virtuel :
```bash
source .venv/bin/activate
```

Vérifiez que l'environnement virtuel est activé en vérifiant la version de Python :
```bash
python --version
```

Pour désactiver l'environnement virtuel, utilisez la commande :
```bash
deactivate
```
